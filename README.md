# Dataset Details
Dataset Description:-

Dataset: UCI Human Activity Recognition (HAR) Dataset
Source: UCI Machine Learning Repository
Total Samples: ~10,000
Features:
Accelerometer (X, Y, Z)
Gyroscope (X, Y, Z)
Activities (Classes):
- Walking
- Walking Upstairs
- Walking Downstairs
- Sitting
- Standing
- Laying

Class Distribution:-
The dataset is fairly balanced across all six activities.
